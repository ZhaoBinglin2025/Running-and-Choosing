# Running-and-Choosing
This is a unity-based interactive system for a more engaging psychological experiment.
When doing psychological experiments, researchers will conduct hundreds even thousands of trials to figure out an effective pattern. 
But this kind of experiment can be boring for both subjects and researchers.Imagine when you are staring 1000+ pictures and you have to be focused or you may miss a trial.
So here it is, a game-like experimental system.
